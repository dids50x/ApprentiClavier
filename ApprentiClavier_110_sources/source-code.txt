Le logiciel ApprentiClavier a été développé sous Visual Basic 4.
Le développement de ce logiciel passe par 3 étapes :

1.  Traitement du projet ApprentiClavier.vbp,
se terminant par la création de l'exécutable ApprentiClavier.exe.
Il est conseillé de mettre la variable FullScreenSwitch à 0 pour le debug.

2. Traitement du Setup qui copiera tous les fichiers nécessaires,
grâce au projet ApprentiClavier_SetUp.vbp,
se terminant par la création de l'exécutable ApprentiClavier_SetUp.exe.

3. Création de l'exécutable d'installation destiné à l'utilisateur,
qui ajoutera les modifications de la base de registre "à la Windows",
qui traitera les fichiers de leçons personnalisées,
et qui appellera ApprentiClavier_SetUp.exe :
utilisez le fichier ApprentiClavier.nsi, que vous compilerez.
Cette étape finale suppose que vous ayez installé le logiciel gratuit NSIS.
